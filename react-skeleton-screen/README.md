
# React Editor Setup

I configured here vs code with eslint and prettier to work with React.


## Installation

Install my-project with npm

```bash
  git clone https://github.com/rohannnagibbtasin13/reactEditorSetup.git
  cd reactEditorSetup
  yarn or npm start
  yarn start or npm start
```
